Política de Privacidade - Minha Super Capivara
Última atualização: 30/04/2026
A sua privacidade é importante para nós. Esta política explica como o aplicativo Minha Super Capivara lida com suas informações.
1. Coleta de Informações
O aplicativo Minha Super Capivara foi desenvolvido para ser um jogo offline.
•
Dados Pessoais: Nós não coletamos, não armazenamos e não compartilhamos nenhuma informação de identificação pessoal (como nome, e-mail, telefone ou localização).
•
Dados de Jogo: O progresso do jogo (nível, moedas, troféus) é salvo localmente no seu dispositivo através do recurso SharedPreferences. Esses dados não saem do seu celular.

2. Permissões do Dispositivo
O app solicita apenas as permissões básicas necessárias para o seu funcionamento:
•
Armazenamento: Para salvar o seu progresso no jogo.
•
Áudio: Para reproduzir os efeitos sonoros e músicas da capivara.

3. Serviços de Terceiros
O app utiliza a biblioteca Google Fonts. Ao carregar fontes, o Google pode receber informações técnicas limitadas (como endereço IP), necessárias para a entrega das fontes.

4. Privacidade de Crianças
Como o app não coleta dados pessoais, ele é seguro para todas as idades, incluindo crianças. Não há rastreamento de comportamento ou anúncios direcionados.

5. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade ocasionalmente. Recomendamos que você revise esta página periodicamente.
